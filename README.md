
# Centralized Digital Healthcare Platform

A centralized healthcare management platform designed to streamline patient management, appointment scheduling, medical records, and healthcare service access through a secure digital ecosystem.

## Features

- Patient Registration & Management
- Appointment Scheduling
- Electronic Medical Records (EMR)
- Healthcare Service Management
- Secure User Authentication
- Responsive User Interface
- Role-Based Access Control
- Digital Healthcare Data Management

## Tech Stack

- React.js
- JavaScript
- Node.js
- Express.js
- MongoDB
- Tailwind CSS
- REST APIs

## Setup Instructions

### Prerequisites

- Node.js
- MongoDB
- Git

### 1. Clone Repository

```bash
git clone <repository-url>
cd Centralized-Digital-Healthcare-Platform
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file and add:

```env
MONGODB_URI=your_mongodb_connection
PORT=5000
JWT_SECRET=your_secret_key
```

### 4. Run Application

```bash
npm run dev
```

Application will start on:

```text
http://localhost:5173
```
